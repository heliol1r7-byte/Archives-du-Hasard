# Archives-du-Hasard
Archive du Hasard : une expérience interactive où la logique pure défie le bon sens. Explorez 5 paradoxes où votre intuition vous trahit. Du célèbre Monty Hall au vertige de Saint-Pétersbourg, manipulez les probabilités et découvrez les failles de votre esprit. Ne cherchez plus à avoir raison, comprenez pourquoi vous avez tort.
